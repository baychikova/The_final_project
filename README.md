## Описание проекта

В рамках проекта выполнены:

1. **Работа с базой данных**  
   - Составлены SQL-запросы для получения данных о курьерах и заказах.
   - Проверена корректность статусов заказов в БД.

2. **Автоматизация теста к API**  
   - Реализован автотест сценария: создание заказа + получение заказа по треку.
   - Использованы `pytest`, `requests`.

3. **Документирование**  
   - Оформлены скриншоты выполнения SQL-запросов.
   - Приложен скриншот успешного запуска автотеста.
